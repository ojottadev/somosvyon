# 🔗 Meus Links

Página de links estilo Linktree, grátis e sem anúncios.

## 🚀 Como usar

1. Edite o `index.html` com seus links e informações
2. Mude as cores no `style.css` se quiser
3. Abra o `index.html` no navegador pra testar
4. Hospede na Vercel, Netlify ou GitHub Pages arrastando a pasta

## 📁 Arquivos

- `index.html` - Estrutura da página
- `style.css` - Estilos
- `script.js` - Contador de cliques

## 🎨 Personalização

- **Avatar**: troque a URL no `src` da imagem
- **Links**: edite os `<a href="...">`
- **Cores**: mude o `background` no CSS
- **Bio**: edite o texto no `<p class="bio">`

## 📊 Contador de cliques

Os cliques são salvos no navegador. Veja no console (F12).
